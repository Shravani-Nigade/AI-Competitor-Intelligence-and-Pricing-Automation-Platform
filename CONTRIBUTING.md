# Contributing

Thank you for your interest in contributing to the AI Competitor Intelligence and Pricing Automation Platform.

Contributions that improve functionality, fix bugs, optimize performance, enhance documentation, or introduce new features are highly appreciated. Please follow the steps below to contribute:

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Write clean, well-documented, and tested code.
4. Commit your changes with meaningful commit messages.
5. Push your branch and submit a Pull Request.

## Contribution Guidelines

- Follow Python PEP-8 coding standards.
- Keep code modular and reusable.
- Document any new functionality.
- Ensure existing functionality remains unaffected.
- Test your code before submitting.

We appreciate every contribution that helps improve the project.
